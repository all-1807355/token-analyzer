# WEBPAGE Documentation

## Project Overview
This project is a simple webpage frontend for a token analyzer that helps in detecting malicious tokens based on a set of metrics. It features a centered search bar with a selector for choosing a "chain" value between Ethereum and Binance Smart Chain.

## How to use
- run the command: docker run -d -p 8000:8000 token-analyzer
- open http://localhost:8000 on a browser of choice
## Features
- Centered search bar for user input.
- Chain selector to choose between Ethereum and Binance Smart Chain.
- Minimalistic design.
