# WEBPAGE Documentation

## Project Overview
This project is a simple webpage frontend inspired by tokensniffer.com. It features a centered search bar with a selector for choosing a "chain" value between Ethereum and Binance Smart Chain. The design follows a minimalistic style with a muted dark tone and purple accents.

## Project Structure
```
WEBPAGE
├── src
│   ├── css
│   │   ├── reset.css
│   │   └── styles.css
│   ├── js
│   │   └── main.js
│   └── images
├── index.html
└── README.md
```

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the webpage.
3. Ensure that all CSS and JavaScript files are correctly linked in the HTML document.

## Features
- Centered search bar for user input.
- Chain selector to choose between Ethereum and Binance Smart Chain.
- Minimalistic design with a muted dark tone and purple accents for a modern look.

## Future Enhancements
